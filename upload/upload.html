UPLOAD

The Upload module allows users to directly upload ASL MRI datasets into the system. Using the QC Toolbox v1.0, the system automatically:

  • Analyzes uploaded MRI data for common artifacts
  • Detects the type and location of each artifact in voxel-level detail
  • Generates visual outputs such as heatmaps and slice overlays highlighting problematic regions
  • Links detected artifacts with structured dictionary entries for documentation
  • Supports automated Quality Control (QC) workflows for research and clinical use

This automated approach enables users to quickly understand **artifact type, severity, and exact location** without manual inspection, improving reproducibility and efficiency in MRI data analysis.
